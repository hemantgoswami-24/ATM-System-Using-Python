# ATM-System-Using-Python
A simple ATM simulation program built using Python OOPs concepts. Features include PIN creation, PIN validation, balance checking, deposit, and withdrawal functionalities with secure access handling.
ATM System Using Python

This project is a simple ATM simulation built using Python OOP concepts.
It supports secure PIN creation and verification, along with banking operations.

Features

Create 4-digit PIN

PIN verification with attempts

Deposit amount

Withdraw amount

Check balance

Private attributes using double underscore

Tech Stack

Python 3

OOP (Classes, Encapsulation, Methods)
